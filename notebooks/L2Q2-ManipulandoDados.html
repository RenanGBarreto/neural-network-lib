Para resolver problemas com Redes Neurais, assim como qualquer problema de aprendizado de mquina, antes de elaborar uma arquitetura necessrio primeiramente entender o problema e, principalmente, compreender os dados. Esse processo extenso e compreende muitas vezes a maior parte do trabalho, de forma que podemos pensar em uma pirmide, em que no topo est a arquitetura da rede de fato e, antes disso, esto os processos de obteno e manipulao de dados, ilustrado na Figura 1. Dessa forma, esse notebook tem como objetivo introduzir alguns pontos importantes da pirmide para obteno de melhores resultados de uma rede neural. O notebook organizado de acordo com seis pontos principais:

  • Definio do problema
  • Entendendo o dataset
  • Identificando problemas
  • Solucionando problemas
  • Correlao
  • Implementando a rede neural

Definição do problema¶

Nessa atividade, ser utilizado como base de dados informaes dos passageiros do Titanic para identificar quais passageiros sobreviveram. No Titanic, uma das razes que causou o naufrgio foi que no havia botes salva-vidas suficientes para os passageiros e a tripulao. Dentre os passageiros, alguns grupos de pessoas tinham maior probabilidade de sobreviver do que outros, como mulheres, crianas e a classe alta. Dessa forma, o problema consiste em utilizar rede neural para identificar quais pessoas poderiam sobreviver.

Identificando problemas¶

Mesmo com uma base de dados previamente definida, muitas vezes existem problemas que no foram tratados nela, sendo necessrio analisar manualmente. Dentre os problemas, para bases de texto, trs so bastante comuns:

  • Instncias com informaes faltando (NaN) para determinados atributos
  • Dados discrepantes e outliers
  • Dados desbalanceados

Dessa forma, deve-se itentificar a presena desses problemas no dataset.

  • Dica 1: utilize funes das bibliotecas do Pandas para os dois primeiros problemas.
  • Dica 2: visualize a distribuio de instncias por classe atravs de bibliotecas grficas de Python (e.g. matplob, seaborn e pyplot).

Implementando a Rede Neural¶

Uma vez que a base de dados foi analisada e possveis problemas foram corrigidos, pode-se implementar a rede neural. Para esse problema, necessrio uma arquitetura multicamada MLP de no mximo 6 camadas escondidas, recomendando-se a implementao das camadas de forma gradual. Da mesma forma, cada camada dever conter no mximo 30 neurnios. Alm disso, recomenda-se a utilizao de tcnicas de regularizao como Dropout, normalizao do batch e normalizao L2.

Utilize o otimizador de sua escolha para o treinamento por batch, aumente a quantidade de epochs e batch tambm gradativamente, no necessrio uma quantidade muito alta para resoluo do problema. Ao final, mostre a curva de treinamento e a matriz de confuso obtida para o problema.
