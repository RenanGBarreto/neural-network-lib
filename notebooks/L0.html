Perceptron¶

Perceptron um classificador binrio linear e representa um neurnio, a estrutura bsica de uma rede neural. No perceptron, recebe-se os atributos de entrada da base de treinamento (e.g. as entradas de uma porta lgica AND/OR) e multiplica, cada uma delas, por um peso W, conforme Figura 1. Feito isso, os valores resultantes so somados e passam por uma funo de ativao. Nesse notebook, todos os passos para implementao do perceptron sero feitos utilizando Numpy, para isso, 5 etapas devero ser feitas:

  1. Inicializaao dos pesos e bias
  2. Implementando funes de ativao
  3. Calculando a sada do neurnio
  4. Predio
  5. Treino e avaliao

Parte 1 - Inicialização dos pesos e bias¶

Ao implementar um perceptron, o primeiro passo iniciar os pesos em um intervalo pequeno, como [-0.5,0.5] aleatoriamente. O bias quando necessrio tambm deve ser inicializado nessa etapa.

Para implementar essa etapa, vo deve utilizar a funo weight_init(num_inputs). Dica: voc pode utilizar a funo random do numpy

In [2]:
def weight_init(num_inputs): 
    """
    Funo que inicializa os pesos e bias aleatoriamente utilizando numpy
    Parmetro: num_inputs - quantidade de entradas X
    Retorna: w,b - pesos e bias da rede inicializados
    """
    w = np.random.rand(num_inputs) - 0.5
    b =  random.random() - 0.5           # O normal seria iniciar com zero.
    return w,b

Parte 2 - Implementação das funções de ativação¶

As funes de ativao definem o intervalo de valores que a sada do neurnio poder ter. Para redes neurais tradicionais, utiliza-se as funes degrau e sigmoid. Redes neurais profundas podem utilizar as funes ReLU, LeakyReLU e Tangente Hiperblica para evitar problemas no gradiente.

Nsse Notebook, as quatro funes de ativao devem ser implementadas, para verificar a corretude das mesmas, a funo visualizeActivationFunc exibe os grficos correspondentes, as funoes, suas respectivas sadas e grfico devero ser similares ao exposto abaixo: (Dica: utilize a funo exp do numpy)

  • Degrau: sada 0 se menor que 0 e sada 1 caso contrrio $$ \begin{equation} degrau =\begin{cases} 1, & \text{se $x>0$}.\\ 0, & \text{caso contrrio}. \end{cases} \end{equation} $$
  • Sigmoid: sada entre [0,1] $$ \begin{equation} sigmoid = \frac{1}{1 + e^{-z}} \end{equation} $$
  • Retificadora (Relu): sada 0 caso entrada seja negativa e maior que 1 caso contrrio $$ \begin{equation} rely = max(0,x) \end{equation} $$
  • Tangente Hiperblica: sada entre [-1,1] $$ \begin{equation} tanh = \frac{2}{e^{-2*z}} - 1 \end{equation} $$

def activation_func(func_type, z):
    """
    Funo que implementa as funes de ativao mais comuns
    Parmetros: func_type - uma string que contm a funo de ativao desejada
                z - vetor com os valores de entrada X multiplicado pelos pesos
    Retorna: sada da funo de ativao
    """
    ### Seu cdigo aqui (~2 linhas)
    if func_type == 'sigmoid':
        return 1/(1 + np.exp(-z))
    elif func_type == 'tanh':
        return math.sinh(z)/math.cosh(z)
    elif func_type == 'relu':
        return 0 if (z<0) else z
    elif func_type == 'degrau':
        return 0 if (z < 0) else  1

Parte 3 - Cálculo da saída do neurônio¶¶

Com os pesos, bias inicializados e a funo de ativao implementada, calcula-se a sada atravs da equao:

$$ \begin{equation} Z = W*X + b \end{equation} $$ Feito isso, a sada final calculada a partir da funo de ativao escolhida. Para implementar essa funo, voc pode utilizar a funo dot do numpy para multiplicar

def forward(w,b,X):
    """
    Funo que implementa a etapa forward propagate do neurnio
    Parmetros: w - pesos
                b - bias
                X - entradas
    """
    z = np.dot(w,X)+b
    out = activation_func('relu', z)
    return out

Parte 4 - Predição¶

De posse da sada, deve-se avaliar o sucesso da mesma definindo-se um limiar. Para problemas binrios, pode-se estabelecer o limiar em 0.5, de forma que abaixo disso a sada 0 e 1 caso contrrio.

def predict(out):
    """
    Funo que aplica um limiar na sada
    Parmetro: y - sada do neurnio
    """
    ### Seu cdigo aqui (~1 linha)
    return 0 if (out < 0.5) else 1

Parte 5 - Treino e Avaliação¶

Durante o treinamento, a sada calculada pela funo propagate n vezes, onde n a quantidade de interaes do algoritmo. Na primeira interao, os pesos possuem valores pr-definidos pela funo de inicializao e so aleatrios aps essa interao, as prximas calculam o peso baseado em um erro, calculado a partir da equao:

$$ \begin{equation} erro = y - ypred \end{equation} $$

Onde y a sada original do conjunto de dados e y_pred as saidas calculadas. Dado o erro, os pesos so atualizados a partir da equao:

$$ \begin{equation} w += erro*taxa-de-aprendizado*X \end{equation} $$

Onde X o conjunto de entrada e a taxa de aprendizagem um parmetro de otimizao que possui seus valorse variando entre [0,1]. Recomenda-se o uso de taxas de aprendizagem medianas para problemas com redes neurais tradicionais simples (como 0.2-0.5) e taxas de aprendizagem menores para redes neurais profundas (acima de 0.02).

def perceptron(x,y, num_interaction, learning_rate):
    """
    Funo que implementa o loop do treinamento 
    Parmetros: x - entrada da rede 
                y - rtulos/labels
                num_interaction - quantidade de interaes desejada para a rede convergir
                learning_rate - taxa de aprendizado para clculo do erro
    """
    training_interation = []
    training_erro = []
    
    #Passo 1 - Inicie os pesos e bias (~1 linha)
    w,b = weight_init(len(x[0]))
    #Passo 2 - Loop por X interaes
    for i in range(num_interaction):
        
        # Ajuda no calculo do erro mdio quadrado
        erros_interacao = []
        
        for j in range(len(x)):
            
            # Passo 3 -  calcule a sada do neurnio (~1 linha)
            y_calc = predict(forward(w,b,x[j]))
            
            # Passo 4 - calcule o erro entre a sada obtida e a sada desejada nos rtulos/labels (~1 linha)
            erro = y[j] - y_calc  
            
            # Adiciona o erro quadrado dessa instancia
            erros_interacao.append(erro*erro)
            
            # Ajusta os pesos e bias
            w = w + erro*learning_rate*x[j]            
            b = b + erro*learning_rate
        
        # Calcula o erro mdio dessa interao
        erro_medio = np.average(erros_interacao)
        
        training_interation.append(i)
        training_erro.append(erro_medio) 
        
        # Apenas para garantir que pare quando o erro atingir um valor limite mnimo
        if (erro_medio < 0.000001):
            print("Finalizado na interao %d visto que o erro mdio j se tornou muito pequeno %f" % (i, erro_medio))
            break;
   
    # Cria um grafico com os dados sobre o erro de treinamento
    plt.plot(training_interation, training_erro)
    plt.xlabel('Interao')
    plt.ylabel('Erro mdio quadrado')
    plt.show()

    # Preenche o array de saidas preditas
    y_pred = []
    for j in range(len(x)):
        y_pred.append(predict(forward(w,b,x[j])))
       
    # Mtricas de Avaliao
    print('Matriz de Confuso:')
    print(confusion_matrix(y, y_pred))
    print('F1 Score:')
    print(classification_report(y, y_pred))
    
    return w, b

def gerar_pontos( N):
    """
    Funo que gera uma dataset com N amostras de forma que o problema se torna linear
    """
    xA,yA,xB,yB = [random.uniform(-1, 1) for i in range(4)]
    V = np.array([xB*yA-xA*yB, yB-yA, xA-xB])
    
    X = []
    Y = []
    for i in range(N):
        x1,x2 = [random.uniform(-1, 1) for i in range(2)]
        x = np.array([1,x1,x2])
        s = int(np.sign(V.T.dot(x)))
        X.append(x)
        Y.append(s if s > 0 else 0)
    return X,Y

X,Y = gerar_pontos(1000)
